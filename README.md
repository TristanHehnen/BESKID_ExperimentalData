# Experimental Data of the BESKID Project

The repository contains the public experimental data of the BESKID project.

The primary material investigate is Poly(methyl methacrylate) (PMMA).

The materials have been examined in various experimental setups, across different sizes, from micro- to real-scale. Data of the following setups is available:
- Thermogravimetric analysis (TGA)
- Micro-scale combustion calorimetry (MCC)
- Differential scanning calorimetry (DSC)
- Cone calorimetry
